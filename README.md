# Inequalytics
Ce rapport cartographie les **inégalités d’accès à l’éducation dans les pays du Sud** à partir des données de la Banque mondiale (2000–2023). Grâce à un **outil interactif Streamlit**, il révèle des écarts persistants entre zones rurales et urbaines, filles et garçons, et appelle à rendre l’éducation réellement universelle.

This report maps **inequalities in access to education across the Global South** using World Bank data (2000–2023). Through an **interactive Streamlit dashboard**, it reveals persistent gaps between rural and urban areas, boys and girls, and calls for **making education truly universal**.
